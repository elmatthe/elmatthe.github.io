# CPI Downloader User Guide

This is the rendered web version of the CPI guide so it displays properly on the site.

## Quick Use

1. Install dependencies: `openpyxl`, `requests`, and `python-dateutil`.
2. Run `cpi_dashboard_downloader-v7.py`.
3. Select your workbook template: **CPI.xlsx**.
4. Choose date range and provide your FRED API key.
5. Click download/write, then open your workbook and review `CPI_Downloader`.
